Overview

Twikey Integration application is integrate Twikey with Odoo. There is an integration of Mandate and Invoices with Odoo.

Authentication

  • - Authentication of Twikey with Odoo


Mandate

  • - Create a Mandate from Odoo

  • - Update Mandate Details on Twikey server as well as in Odoo.


Invoice

  • - Create Invoice from sales order in Odoo and same will be create on Twikey

  • - Update Invoice Details on Twikey server as well as in Odoo.
