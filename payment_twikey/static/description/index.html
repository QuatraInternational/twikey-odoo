The Twikey module allows usage of the Twikey payment gateway within Odoo. Currently it allows generation and registration of Sepa Direct Debit Mandates as well as payments of Invoices generated within Odoo.

Configuration

Configuration of Twikey with Odoo is easy as pie, just pass in your API key and you're ready to roll.


Creation of Mandates

Creation of Mandates from within Odoo via invites to your customers. They are automatically updated in Odoo when updates are made in Twikey via BankSwitching or other mechanisms.


Payment of Invoices

Invoices created from sales orders will include a QR code and sent to Twikey for payment. Allowing either one time payment of recurring for automatic payment of future invoices. Updates about (paid) invoice are almost real-time visible within Odoo.
